## Edux
```
cd edux
npm install
node ./filename
```
Programy rozwiązujące przykłądy z Eduxa - stąd strktura kodu analogiczna jak w nim.
Programy nie przyjmują argumentów - dane są zahardkodowane - należy zmienić skrypt i uruchomić odpowiedni plik w Node.
